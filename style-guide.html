001

Design Philosophy

A disciplined, type-driven system built around readability, grid structure, and a minimal monochromatic palette. Every element earns its place.

01
Type-Driven Hierarchy

Two typefaces do all the work. Resistance handles display, labels, and timestamps with authority. At Textual carries the reading experience. Size, weight, and letter-spacing create hierarchy without color.

02
Grid as Structure

CSS Grid with 1px gaps creates the visual skeleton. The gap color is the page background — a single design choice that unifies every layout. Columns, rows, and sidebars all emerge from this one pattern.

03
Black, White, Gray

Five colors. No accent, no gradient, no shadow. High contrast for legibility, gray for depth and hierarchy. If a new app needs a color, it should be used sparingly and with intention.

002

Color System

Five tokens. Black and white anchor the contrast. Three grays carry the mid-range — backgrounds, borders, and secondary text. Semantic intent matters more than the values.

--black #111111 Headers, body text, active outlines, badges, borders
--white #FFFFFF Cell backgrounds, card surfaces, text on dark backgrounds
--gray-1 #F4F4F4 Hover states, expand panel background, alternate sections
--gray-2 #E8E8E8 Grid gap, dividers, borders, page background
--gray-3 #AAAAAA Secondary text, labels, metadata, timestamps, placeholders

004

Spacing & Scale

Based on an 18px root. All spacing in rem. The 1px grid gap is a design signature — keep it. Content padding uses the 0.75–1.5rem range. Sections breathe at 1.5–2rem.
